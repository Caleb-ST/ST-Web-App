Daily Workflow
  1. Go to Setup → Connect Google Sheets
  2. Run Enquiries & Sales Actions Report (ERP via RDP)
  3. Copy ALL the report text
  4. Select Store + date range in Import tab
  5. Paste → Parse → review preview → Send to Sheet
Status Rules
All note lines combined and checked against every rule. Highest priority match wins. Use the 🧪 Test box to debug rules instantly.
Auto-Complete Toggle
Only tick when the report covers ALL active jobs in that date window. Jobs outside the window are never touched regardless.
Undo Last Send
Restores all previous statuses and clears newly added rows. One level of undo only. Click before doing anything else.
Google Auth
Token lasts 1 hour per session. If you get auth errors while working, go to Setup → Connect Google Sheets again. No data is lost.
